# CookingCraftingandCropRaisingContainers
Cooking, Crafting and Crop Raising Containers used for 1931 Bootleg RP server

## How it works
  Using Unturned's Point Gesture, point at the configured container with ingredients and the plugin will start the process of creating the recipe.<br>
  After a configurable amount of time, the recipe will be made and ingredients replaced in the container.

## Commands
  No commands necessary. All done via the in-game point gesture.
